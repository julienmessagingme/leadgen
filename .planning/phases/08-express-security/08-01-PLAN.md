<task type="auto">

  <files>src/api/auth.js</files>
  <action>
1. In `src/api/auth.js`:
   - Add `const rateLimit = require("express-rate-limit")` at top
   - Create login rate limiter (SEC-01):
     ```js
     const loginLimiter = rateLimit({
       windowMs: 15 * 60 * 1000, // 15 minutes
       max: 10,
       message: { error: "Too many login attempts, please try again later" },
       standardHeaders: true,
       legacyHeaders: false,
     });
     ```
   - Apply to login route: `router.post("/login", loginLimiter, async (req, res) => { ... })`
   - Change JWT expiry from `"7d"` to `"24h"` (AUTH-01)
   - Change JWT sub from `email` to `"admin"` (AUTH-02): `jwt.sign({ sub: "admin" }, ...)`
  </action>
