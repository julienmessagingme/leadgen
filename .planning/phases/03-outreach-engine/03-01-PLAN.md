**MessagingMe API wrapper (create new src/lib/messagingme.js on VPS):**
- Base URL: `https://ai.messagingme.app/api`
- Internal `messagingme(endpoint, body)` helper: POST with Content-Type application/json, X-API-Key from MESSAGINGME_API_KEY env, X-Workspace-Id from MESSAGINGME_WORKSPACE_ID env. Throws descriptive error on non-ok response.
- `createWhatsAppTemplate(name, bodyText, buttonUrl)` -- POST /whatsapp-template/create with name, language 'fr', category 'MARKETING', components array: BODY with text, BUTTONS with URL button "Prendre RDV".
- `listTemplates(name)` -- POST /whatsapp-template/list with optional name filter. Returns templates array.
- `syncTemplates()` -- POST /whatsapp-template/sync with empty body. Syncs Meta status.
- `sendWhatsAppByUserId(userId, templateNamespace, templateName, lang, params)` -- POST /subscriber/send-whatsapp-template-by-user-id with user_id, create_if_not_found 'yes', content object with namespace, name, lang (default 'fr'), params.
- Lazy pattern: do NOT throw at module load if env vars missing. Check in messagingme() helper and throw there.

**Claude Sonnet message generator (create new src/lib/message-generator.js on VPS):**
- Import `{ anthropic }` from ./anthropic.js.
- Use `anthropic.beta.messages.create` with `output_config: { format: { type: 'json_schema', schema: ... } }` -- consistent with existing icp-scorer.js beta path.
- Model: `claude-sonnet-4-6-20250514` for all message generation.
- 5 exported functions, each with its own structured schema and prompt:

1. `generateInvitationNote(lead)` -- returns string (max 280 chars). Schema: `{ note: string }`. Prompt: Tu es Julien Poupard, DG de MessagingMe. Invitation LinkedIn personnalisee. Reference au signal detecte. Ton pro mais humain. Max 280 caracteres STRICT. Pas d'emojis, pas de pitch. HARD LIMIT: truncate to 280 chars after generation with `substring(0, 277) + '...'` if over.

2. `generateFollowUpMessage(lead)` -- returns string. Schema: `{ message: string }`. Prompt: Message de suivi post-connexion LinkedIn. Remercier pour la connexion, proposer un echange sur le sujet du signal, mentionner MessagingMe brievement. 3-5 phrases max.

3. `generateEmail(lead)` -- returns `{ subject, body }`. Schema: `{ subject: string, body: string }`. Prompt: Email de relance J+7. Objet accrocheur. Corps HTML: reference a la connexion LinkedIn, proposition de valeur MessagingMe, CTA Calendly (URL from CALENDLY_URL env var). Signature Julien Poupard.

4. `generateWhatsAppBody(lead)` -- returns string. Schema: `{ body: string }`. Prompt: Corps WhatsApp 3-4 lignes. Reference au signal et a l'echange LinkedIn. Proposition RDV via Calendly. Ton direct et personnel.

5. `generateInMail(lead)` -- returns `{ subject, body }`. Schema: `{ subject: string, body: string }`. Prompt: InMail LinkedIn complet. Objet percutant. Corps: reference au signal, valeur MessagingMe pour leur secteur, CTA.

- Each function: try/catch wrapping the Anthropic call. On error, log warning and return a fallback (null for note/message, null for email/inmail objects). Let the calling task decide what to do with null.
