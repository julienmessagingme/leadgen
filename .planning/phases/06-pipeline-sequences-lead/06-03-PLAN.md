- Use `useLead(selectedLeadId)` hook for drawer detail data

**Layout (top to bottom):**
1. NavBar (from shared components)

3. FilterBar (tier + source + search, showStatus=false since kanban shows status visually)
4. Conditional render: KanbanBoard if view==="kanban", ListView if view==="list"
5. LeadDrawer (conditionally open when selectedLeadId is set)

**ViewToggle** (inline component or extracted):
- Per user decision: "Tab-style toggle" with text labels "Kanban" and "Liste"
- Inline-flex with rounded-lg bg-gray-100 p-1, active tab gets bg-white shadow-sm

**Drawer integration:**
- When a lead card/row is clicked, set selectedLeadId
