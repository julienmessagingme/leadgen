**Sequences.jsx** -- Main sequences page:

**State management:**
- `filters` state: `{ status: "", tier: "", source: "", search: "" }` -- includes status filter (unlike pipeline)
- `sort` state: `{ field: "icp_score", order: "desc" }` -- per user decision: "Default sort: ICP score descending"
- `selectedLeadId` state: null or lead ID (for drawer)
- `selected` state: `new Set()` -- multi-select for bulk actions
- `showConfirm` state: `{ open: false, action: "", count: 0 }` -- for bulk exclude confirmation

**Data fetching:**
- Use `useLeads({ ...filters, sort: sort.field, order: sort.order })` hook
- Use `useLead(selectedLeadId)` for drawer detail
- Use `useLeadAction()` for individual actions from drawer or table
- Use `useBulkAction()` for bulk operations

**Selection management:**
- Implement toggle/toggleAll/clear functions (per RESEARCH.md multi-select pattern)
- CRITICAL: Clear selection when filters change (per RESEARCH.md Pitfall 7 -- selection state diverges from displayed data)
- Use useEffect watching filters to call clear()

**Layout (top to bottom):**
1. NavBar
2. Page header: "Sequences" title + sort indicator text
3. FilterBar (with showStatus=true -- sequences needs status filtering)
4. SequenceTable with leads data, selected set, sort state, callbacks
5. BulkActionBar (visible when selected.size > 0)
6. LeadDrawer (when selectedLeadId is set)
7. ConfirmDialog (for bulk exclude or individual exclude from table)

**Sort interaction:**
- Clicking a sortable column header toggles: if same field, flip order. If different field, set desc.
- Visual indicator on sorted column header (arrow up/down)

**Individual actions from table rows:**
- Pause/resume: call `useLeadAction.mutate({ id, action })` directly (instant, per user decision)
- Exclude: show ConfirmDialog first, then call mutation on confirm

**Bulk actions:**
- Pause/Resume: call `useBulkAction.mutate({ ids: [...selected], action })` directly
- Exclude: show ConfirmDialog with count: "Exclure {N} lead(s) ? Cette action est irreversible." On confirm, call mutation then clear selection.

**Drawer integration:**
- Same as Pipeline: click lead row opens drawer with all 5 sections
- ActionButtons in drawer use same individual action handlers

**Loading/error/empty states:**
- Loading: show skeleton rows
- Error: show error card with retry
- Empty: "Aucun lead en sequence d'outreach"
