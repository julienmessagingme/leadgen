**FilterBar.jsx** -- Inline filter bar with dropdowns and search:
- Props: `filters` (object with status, tier, source, search), `onChange` (callback receiving updated filters), `showStatus` (boolean, default true -- sequences needs status filter, pipeline may not)

- Tier dropdown: "Tous tiers" / "Hot" / "Warm" / "Cold"
- Source dropdown: "Toutes sources" / "concurrent" / "influenceur" / "sujet" / "job"
- Status dropdown (conditional on showStatus prop): "Tous statuts" / key statuses
- Search input: text field with placeholder "Rechercher par nom ou entreprise...", min-w-64
- Search uses React 19 `useDeferredValue` for debounce (per RESEARCH.md "Don't Hand-Roll" section)
- Each change calls `onChange({ ...filters, [field]: value })`
- Export default

- Focus trap not needed (simple modal for single use case)
- Export default
