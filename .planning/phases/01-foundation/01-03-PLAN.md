2. Create `/home/openclaw/leadgen/src/lib/logger.js`:
   - Import supabase client from `./supabase`
   - `async function log(runId, task, level, message, metadata = null)`:
     - Insert into `logs` table: { run_id, task, level, message, metadata, created_at }
     - level must be one of: 'debug', 'info', 'warn', 'error'
     - On insert failure: `console.error('Log write failed:', error.message)` — NEVER throw from logger
     - This is critical: the logger must catch its own errors to prevent infinite error loops
   - `async function logTaskRun(runId, task, status, errorMsg = null)`:
     - Convenience wrapper: calls log() with appropriate level (error for 'error' status, info otherwise)
     - Message format: `Task ${task} ${status}`
   - Export: { log, logTaskRun }

3. Create `/home/openclaw/leadgen/src/lib/suppression.js`:
   - Import crypto (built-in) and supabase client
   - `function hashValue(value)`: returns SHA256 hex digest of `value.toLowerCase().trim()`
   - `async function isSuppressed(email = null, linkedinUrl = null, phone = null)`:
     - Hash each non-null value
     - Query `suppression_list` table: `.select('id').in('hashed_value', hashes).limit(1)`
     - If query fails: return true (fail-safe — treat as suppressed if check fails)
     - If data.length > 0: return true (suppressed)
     - Else: return false
   - Export: { isSuppressed, hashValue }
